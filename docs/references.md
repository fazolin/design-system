# references

genealogia visual e conceitual. cada entrada lista o que especificamente influencia este sistema — não admiração genérica, especificidade.

## glitch e dado corrompido

**Rosa Menkman** — *A Vernacular of File Formats* (2010), *The Glitch Moment(um)* (2011). teorização do glitch como linguagem própria, não erro a ser corrigido. influencia: tratar RGB-split, datamosh e block displacement como **vocabulário** com gramática, não como filtros aleatórios.

**Kim Asendorf** — *Mountain Tour* (2010), *Pixel Sorting Tools*. pixel sorting algorítmico. influencia: a ideia de que corrupção é determinística e estética emerge do algoritmo, não do acaso.

**Takeshi Murata** — *Monster Movie* (2005), *Untitled (Pink Dot)* (2007). datamosh como forma plástica. influencia: a estética datamosh como deliberada, fria, estrutural — não nostálgica.

**JODI.org** — site, 1995–presente. interfaces que se recusam a serem usáveis no sentido tradicional. influencia: a hostilidade controlada do design — recusa do convidativo.

## som, dado, minimalismo

**Ryoji Ikeda** — *test pattern*, *data.tron*, *supercodex*. matemática de altíssima resolução, frequência pura, preto e branco com pontuações de cor signaling. influencia direta: paleta restrita (preto + branco + cor de acento), tipografia HUD pequena, dado como conteúdo único.

**Carsten Nicolai (Alva Noto)** — *Unitxt*, *Xerrox* series. visual de osciloscópio, sinal puro, type mono fina. influencia: confiança no minimalismo extremo — uma única linha pode ser a obra inteira.

## generativo e código como medium

**Casey Reas** — *Process* series, co-criador do Processing. arte como execução de processo definido. influencia: shaders standalone, comentados em inglês, com uniforms padronizados — código é artefato auditável, não caixa preta.

**Aaron Koblin** — *House of Cards* (com Radiohead, 2008), *The Sheep Market*. dado real renderizado como retrato. influencia direta: o feed da câmera não é visual — é o sujeito. point cloud de rosto não é efeito — é a obra.

**Cory Arcangel** — *Super Mario Clouds* (2002), *Self-Playing Sony Playstation 1*. apropriação minimalista, dispositivo como artefato. influencia: se a tecnologia é a obra, mostre o dispositivo, não o decore.

## vídeo e desmaterialização

**Nam June Paik** — *Magnet TV* (1965), *Zen for TV* (1963). interferência intencional no sinal de TV. influencia conceitual: o feedback elétrico como gesto autoral. todo o vocabulário de glitch deste sistema deriva, em última instância, daqui.

**Radiohead — *House of Cards*** (2008, dir. James Frost & Aaron Koblin). nenhuma câmera, só dado de point cloud. influencia direta: o material bruto do scanner como acabamento final.

## tipografia e diagramação

**Peter Saville** — *Unknown Pleasures* (Joy Division, 1979). dado científico (pulsar) como capa. influencia: dado bruto pode ser composição. nada de "estilizar" o dado.

**Wim Crouwel** — *new alphabet* (1967). tipografia funcional levada ao limite, leitura como grade computacional. influencia: confiança em tipo mono uppercase como linguagem completa, sem ornamento.

## web

**Rafaël Rozendaal** — *www.intothepool.com*, *www.ifnowistheanswer.com*. cada work é um domínio próprio, infinitamente looping, sem navegação. influencia: cada work fazolin é potencialmente um endereço próprio — coleção de domínios, não sub-rotas de portfolio.
